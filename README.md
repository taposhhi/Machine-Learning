What is Machine Learning?

Machine Learning (ML) হলো Artificial Intelligence (AI)-এর একটি অংশ, যেখানে কম্পিউটারকে স্পষ্টভাবে প্রতিটি নিয়ম লিখে না দিয়ে ডেটা থেকে নিজে নিজে শেখার সুযোগ দেওয়া হয়।
অর্থাৎ, ML মডেল ট্রেনিং ডেটার ভেতরের প্যাটার্ন শিখে নেয় এবং সেই শেখা জ্ঞান ব্যবহার করে নতুন ডেটার ওপর সিদ্ধান্ত বা পূর্বাভাস দিতে পারে।

আজকের প্রায় সব আধুনিক AI সিস্টেমের মূল ভিত্তি হলো মেশিন লার্নিং।
যেমন:

আবহাওয়া বা স্টক মার্কেট পূর্বাভাস

স্বয়ংচালিত গাড়ি (self-driving cars)

Large Language Models (LLMs)

Generative AI টুলস

সবগুলোর পেছনেই কোনো না কোনো ML মডেল কাজ করছে।

Machine Learning কীভাবে কাজ করে?

মেশিন লার্নিং-এর মূল ধারণা খুব সহজ:

যদি কোনো মডেলকে এমন ডেটা দিয়ে ট্রেন করা হয়, যা বাস্তব জীবনের সমস্যার মতো দেখতে,
তাহলে সেই মডেল নতুন ডেটা দেখেও সঠিক সিদ্ধান্ত নিতে পারবে।

এই শেখার প্রক্রিয়াকে বলা হয় model training।

কিন্তু ট্রেনিং নিজেই চূড়ান্ত লক্ষ্য না।
আসল লক্ষ্য হলো generalization —
অর্থাৎ ট্রেনিং ডেটা থেকে শেখা জ্ঞান বাস্তব জীবনের সমস্যায় কাজে লাগানো।

যখন ট্রেন করা মডেল বাস্তবে ব্যবহার করা হয় এবং নতুন ইনপুট দেখে আউটপুট দেয়,
তাকে বলা হয় AI inference।

Deep Learning কী?

Deep Learning হলো মেশিন লার্নিং-এর একটি উন্নত অংশ, যেখানে খুব বড় এবং গভীর Artificial Neural Network ব্যবহার করা হয়।

Traditional ML-এ যেখানে মানুষের বানানো ফিচার ও নিয়ম বেশি গুরুত্বপূর্ণ,
Deep Learning সেখানে নিজে নিজেই জটিল ডেটার সূক্ষ্ম প্যাটার্ন শিখে নিতে পারে।

Deep Learning জনপ্রিয় হওয়ার প্রধান কারণ:

প্রচুর ডেটা (Big Data)

শক্তিশালী কম্পিউটিং (বিশেষ করে GPU)

আজকের প্রায় সব state-of-the-art AI মডেল—কম্পিউটার ভিশন, ভয়েস, LLM—সবই Deep Learning ভিত্তিক।

Machine Learning এবং Data Science

Machine Learning আর Data Science একে অপরের সঙ্গে খুব ঘনিষ্ঠভাবে যুক্ত।

সহজভাবে বলা যায়:

Data Science → ডেটা বিশ্লেষণ করে ইনসাইট বের করা

Machine Learning → সেই ইনসাইট ব্যবহার করে স্বয়ংক্রিয়ভাবে কাজ করা

Machine Learning মূলত এমন অ্যালগরিদমের সমষ্টি, যেগুলো ডেটা বিশ্লেষণ শেখে এবং সেই শেখা জ্ঞান দিয়ে নিজে নিজে কাজ সম্পন্ন করে।

Machine Learning শব্দের ইতিহাস

“Machine Learning” শব্দটি জনপ্রিয় হয় ১৯৫৯ সালে
Arthur L. Samuel-এর একটি গবেষণাপত্রের মাধ্যমে,
যেটি প্রকাশিত হয়েছিল IBM Journal-এ।

তিনি সেখানে বলেন, মেশিন লার্নিং-এর লক্ষ্য হলো:

একটি কম্পিউটার এমনভাবে শেখা, যেন সে প্রোগ্রাম লেখকের চেয়েও ভালোভাবে একটি খেলা খেলতে পারে।

এই উদাহরণে তিনি চেকার্স (Checkers) খেলার কথা বলেছিলেন।

Machine Learning vs Artificial Intelligence

অনেকে AI আর Machine Learning-কে এক জিনিস মনে করে, কিন্তু আসলে তারা এক নয়।

সংক্ষেপে:

সব Machine Learning হলো AI

কিন্তু সব AI Machine Learning নয়

Artificial Intelligence বলতে বোঝায় যেকোনো সিস্টেম, যা মানুষের সরাসরি হস্তক্ষেপ ছাড়াই সিদ্ধান্ত নিতে পারে।

Rules-based AI (Traditional AI)

সবচেয়ে সহজ AI হলো rules-based system।

উদাহরণ: একটি থার্মোস্ট্যাট

IF temperature < 67 → heater ON  
IF temperature > 72 → AC ON


এখানে সব নিয়ম মানুষ আগেই লিখে দিয়েছে।

এমনকি জটিল মেডিকেল expert system-ও নিয়মভিত্তিক হতে পারে, যেখানে ডাক্তাররা আগে থেকেই সিদ্ধান্তের গাছ (decision tree) তৈরি করে দেন।

Machine Learning-based AI

Machine Learning-এ নিয়ম মানুষ লিখে দেয় না—মডেল নিজেই শেখে।

উদাহরণ: Spam Email Filter

Rules-based AI → মানুষকে নিজে নিজে spam-এর নিয়ম বানাতে হবে

Machine Learning → শুধু ইমেইলের ডেটা আর অ্যালগরিদম দিলেই হবে

ট্রেনিং-এর সময়:

মডেল ইমেইল দেখে অনুমান করে spam না normal

ভুল হলে error হিসাব করা হয়

অ্যালগরিদম ঠিক করা হয়

এই প্রক্রিয়া বারবার চলতে থাকে

শেষে মডেল নিজেই শিখে ফেলে কোন ইমেইল spam।
